# Kakara Core

The API/Mod API core for Kakara.


```xml
        <repository>
            <id>kakara-repo</id>
            <url>https://repo.kingtux.me/storages/maven/kakara</url>
        </repository>

                <dependency>
                    <groupId>org.kakara</groupId>
                    <artifactId>core</artifactId>
                    <version>1.0-SNAPSHOT</version>
                </dependency>
        ```
